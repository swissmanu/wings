dragStarted(absolutePosition)

_self.dragStarted = function dragStarted(absolutePosition)

Called by the CanvasWrapper, this method looks for the deepest Widget on the given absolute position and sends a native_dragStart event.  The Widget itself can use the events acceptCallback to start a drag and drop sequence.
