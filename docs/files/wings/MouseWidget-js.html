MouseWidget

MouseWidget is a specialization of a simple Widget.  In addition, the MouseWidget reacts on and handles mouse inputs.

MouseWidget is not usable directly.  It is a superclass for Widgets which are reactive on mouse input.

mouse:dragstart

Handles the native dragStart event.  If this MouseWidget is draggable, a “dragRequest” event is emitted.  The implementing MouseWidget/subclass can react on this event and calls one of the two callbacks to start or cancel the drag and drop procedure.

Events

dragRequest

Signals that the user wants to initiate a drag and drop process.  The receiving MouseWidget can use one of the callbacks to accept or cancel the request.

Parameters

(Function) acceptCallbackCallback to call, if a Widget wants to start a drag and drop process
(Fucntion) cancelCallbackCallback to not start a drag and drop or cancel a drag and drop process
